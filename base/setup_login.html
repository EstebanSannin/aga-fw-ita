Device Login

In order to secure the router against unauthorized access, please enter a password for the “admin” user. You will need to enter the user name and password in order to access the router in the future, so you may want to write them down.

U.S. Robotics recommends using a password of eight or more alphanumeric characters (such as “Canyon79Snow” or “2Lake6Forest3”). Please note that the user name and password are case-sensitive.

When you’re done and are ready to apply all your settings, press Next.
